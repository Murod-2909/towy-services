# Security Policy

## Supported Versions

This repository holds a single, continuously deployed website — there are no
maintained older versions. Security fixes are applied to the `main` branch
and deployed as soon as they're merged.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
privately rather than opening a public issue:

- Preferred: use GitHub's [private vulnerability reporting](../../security/advisories/new)
  for this repository.
- Alternatively, email the maintainer at **support@247towy.com** with a
  description of the issue, steps to reproduce, and its potential impact.

Please do not disclose the issue publicly until it has been investigated and
a fix has been released.

We aim to acknowledge new reports within a few business days.
